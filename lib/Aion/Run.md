# SYNOPSIS

```perl
use Aion::Run;

my $aion_run = Aion::Run->new();
```

# INSTALL

Add to **cpanfile** in your project:

```cpanfile
requires 'Aion::Run',
    git => 'https://github.com/darviarush/perl-aion-run.git',
    ref => 'master',
;
```

And run command:

```sh
$ sudo cpm install -gvv
```
